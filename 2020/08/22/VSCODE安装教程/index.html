VSCODE三系统安装教程
发表于2020-08-22|更新于2020-08-30|开发教程
|阅读量:

简介

首先明确一点，VScode是开发应用程序等项目的基础编辑器，是Microsoft（微软公司）的产品，可以运行在Windows、Linux、Mac上使用，支持大多标记，编程语言（插件要求），安装插件后，就可以有智能提示，UI美化等功能，当然市场上还有很多其他的开发工具，在这里我只介绍Visual studio code（VSCode）的安装。

不同系统的安装方法

WINDOWS下的安装

1.首先下载vscode安装软件 https://code.visualstudio.com/download

2.选择vscode版本（Windows 建议user-installer）

3.把vscode安装文件准备好（六十四位电脑选择X64即64bit，三十二位电脑请选择X86即32bit）

4.傻瓜式安装，全为简体中文，直到完成。

5.环境变量（请勾选，安装完成需重启自动配置path）

6.安装成功（到此步骤已经安装成功，在之前的安装过程里可以选择是否创建开始菜单栏和快捷方式，如果没有创建直接到步骤一所说安装目录找code.exe启动）

LINUX下的安装

1.首先下载linux-vscode安装文件 https://code.visualstudio.com/Download

2.选择vscode版本（根据系统基于方式选择，Kali等基于debian系可以选择.deb;rpm系可以选择.rpm）

这里选择.tar.gz包作为演示，其他包的安装方式请查看该系统安装文档。
3.下载一个tar包，然后复制到linux的磁盘上，大部分系统右键菜单都有–>提取到此处（也可以cd到该目录使用tar -xvzf XXXXXXXX.tar.gz解压）

4.进入文件的目录，可以直接双击code可执行文件直接运行Visual Studio Code（也可在终端输入code启动vscode）


进入成功后如图：

MAC下的安装

1.首先下载Mac-vscode安装文件 https://code.visualstudio.com/Download

2.选择macOS

3.下载成功后，在下载项将会多出一个ZIP格式的文件，双击解压。

4.解压后，会得到一个APP格式文件，直接可以运行，建议把它移动到应用程序目录或实用工具目录下。
5.利用终端命令行启动vscode（！！！可选操作！！！）
①修改vim ~/.bash_profile
②添加export PATH=/Applications/Visuall\ Studiol\ Code.app/Contents/Resources/app/bin:$PATH
③重启 source ~/.bash_profile
④终端下输入code（在某一文件或目录）

VSCODE的简单配置

插件利用和推荐

三个系统的插件目录大都一致，直接介绍安装方法

插件利用和搜索

搜索需要插件。win和mac插件图标为四个小方块，linux可在菜单栏的view–>extensions找到

插件推荐：

优雅中国人必备

简体中文语言包 Chinese（simplified）language…….

前端常用插件

Auto Close Tag；Auto Rename Tag；Beautify；Bracket Pair Colorizer；Debugger for Chrome；ESLint……

文件图标：vscode-icons

vscode-icons插件可以实现对各种文件类型的文件前的图标进行优化显示，这样我们在查看长长的文件列表的时候，可以直接通过文件的图标就可以快速知道文件的类型，而不是去看文件的后缀。

暗色主题：One Dark Pro

长时间的撸码，暗色调的编码环境更不容易让视力疲劳，而且也可以让自己更加专注。安装了One Dark Pro 插件后，可以一键将VSCode编辑器的颜色调整成暗色系，编码起来更加舒适，界面更加美观。

代码美化：Beautify

Beautify插件可以快速格式化并高亮你的代码格式，让你在编写代码时杂乱的代码结构瞬间变得非常规整，代码强迫症必备，较好的代码格式在后期维护以及他人阅读时都会有很多的便利。插件支持的语言非常多，基本覆盖了目前所有vscode支持的语言，而且你还可以自定义代码格式化的结构。

代码检查工具：ESLint

ESLint是一个语法规则和代码风格的检查工具，可以用来保证写出语法正确、风格统一的代码。而 VSCode 中的ESLint插件就直接将ESLint的功能集成好，安装后即可使用，对于代码格式与规范的细节还可以自定义，并且一个团队可以共享同一个配置文件，这样一个团队所有人写出的代码就可以使用同一个代码规范，在代码签入前每个人可以完成自己的代码规范检查。

万能语言运行环境：Code Runner

如果你需要学习或者接触各种各样的开发语言，那么Code Runner插件可以让你不用搭建各种语言的开发环境，直接通过此插件就可以直接运行对应语言的代码，非常适合学习或测试各种开发语言。
支持的语言有：C, C++, Java, JavaScript, PHP, Python, Perl, Perl 6, Ruby, Go, Lua, Groovy, PowerShell, BAT/CMD, BASH/SH, F# Script, F# (.NET Core), C# Script, C# (.NET Core), VBScript, TypeScript, CoffeeScript, Scala, Swift, Julia, Crystal, OCaml Script, R, AppleScript, Elixir, Visual Basic .NET, Clojure, Haxe, Objective-C, Rust, Racket, AutoHotkey, AutoIt, Kotlin, Dart, Free Pascal, Haskell, Nim, D, 以及一些自定义的命令。

代码拼写检查：Code Spell Checker

此插件安装后就不用管就好了，在你代码中有单词拼写错误时，你就会发现它的好处，因为我们写代码毕竟都是大量的英文单词变量定义，插件还可以给出错误拼写单词的建议。

备忘插件：TODO Highlight

在很多的其他代码编辑器中都有TODO标注功能的，如你写到某一部分的代码时，其中部分的功能需要稍后再来实现，这是就可以在对应的代码处添加一个TODO类型的注释，那么后期就可以快速地跳转到这部分继续写，而且当项目很大的时候，TODO就变得更加有用，因为有时候TODO可能有几十个，帮助你标注那些功能需要继续实现或优化。
